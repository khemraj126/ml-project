# 🎬 Movie Recommendation System

## 📌 Overview

This project is a **Movie Recommendation System** that suggests movies to users based on their preferences. It uses machine learning techniques like **content-based filtering** and **cosine similarity** to find similar movies.

---

## 🚀 Features

* Recommend movies based on user input
* Uses cosine similarity for accurate matching
* Simple and easy-to-understand implementation
* Scalable for large datasets

---

## 🧠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

---

## 📂 Dataset

This project uses a movie dataset (e.g., MovieLens) containing:

* Movie titles
* Genres
* Ratings

---

## ⚙️ How It Works

1. Data preprocessing and cleaning
2. Feature extraction (genres, keywords)
3. Convert data into vectors
4. Compute similarity using cosine similarity
5. Recommend top similar movies

---

Enter a movie name, and the system will recommend similar movies.

---

## 📊 Example

Input:

```
Inception
```

Output:

```
Interstellar  
The Prestige  
Shutter Island  
```

---

## ⚠️ Limitations

* Cold start problem (new users/movies)
* Depends on dataset quality
* Limited personalization without user history
